# egg-nuonuo

诺诺发票SDK [node-nuonuo] Egg.js插件。

## 安装插件

```bash
npm i @axolo/egg-nuonuo --save
```

## 开启插件

```js
// config/plugin.js
exports.nuonuo = {
  enable: true,
  package: '@axolo/egg-nuonuo',
};
```

## 详细配置

```js
// {app_root}/config/config.default.js
exports.nuonuo = {
};
```

请到 [config/config.default.js](config/config.default.js) 查看详细配置项说明。
